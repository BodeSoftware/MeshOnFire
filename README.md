# MeshOnFire
Niagara System to set meshes on fire in Unreal Engine 5.6

## Overview
Niagara System to set meshes on fire.

## Installation
Drop .uasset into a 5.6 project

## Requirements
- Unreal Engine 5.6
- Niagara Fluids plugin enabled
